DCO 增强人与人的协作

中心化组织

在中心化组织中，个体一般通过签署雇佣协议的方式与中心组织建立雇佣关系，关系的建立和结束都设置了重重障碍，且雇佣期内个体不得服务其他组织，丧失了自由；另一方面，中心化组织对时间、地域等方面的要求较高，限制了个体的参与。

分布式协作组织

CODE币 * 的引入，使得个体间可以通过CODE币建立关系，并用智能合约记录在区块链中，个体可自由决定协作关系的开始和结束，可同时建立任意数量的协作关系，自由度很高；而且CODE币可将不同地域、符合要求的个体组织起来，开展项目协作，资源配置更加优化。

我们把这种基于CODE币建立的组织称为分布式协作组织（Distributed Collaboratiove Organization，DCO）。

*注： CODE 币是整个 DCO Network 内交易媒介、价值计量、债务清偿、价值存储、投资工具、合同延期支付的基础本位币。

CODE 如何在 DCO 中发挥作用

  • 治理

    DCO Network 将提供一系列去中心化的基础设施，包括工具、服务和智能合约，使得创建、管理理、经营一个 DCO 组织变得轻而易举，并且每一个 DCO 可创建自己的治理体制和具体规则。

  • 公开

    区块链提供诚实可信的记账技术，DCO 通过智能合约进行创建，信息全部上链，用户可获得充分的信息以决定加入或退出某个 DCO，投资者也可获得充分的信息以决定是否进行项目投资。

  • 权益

    每一个 DCO 可创建自己的“主通证”（ DCO Token，DCOT ) 和其他高阶通证（ High Order Token, HOT )，使用通证为每一个人的贡献进行公平的衡量，以此为依据进行强有力的经济激励。

  • 流通

    DCO Network 将提供交易能力，撮合 DCO 币与任何一个 DCOT 的交易。 DCO 币作为平台的基础货币，将登陆主流加密数字货币交易所，与比特币、以太币等数字货币建立汇率联系。

DCO Network 是 DCO 的支撑平台

“ DCO Network 是一个面向全球的协作平台，将提供一组技术标准、工具和基础设施，支持 DCO 的创建、运行和发展。"

  • 服务

    在产品服务层面，DCO Network 将结合第三方成熟SAAS服务，以通证为纽带，为DCO提供通证管理SAAS服务，帮助有创意的创业者凝聚自己的社群。

  • 工具

    在管理工具层面，DCO Network 将为DCO提供各种便捷的组织管理工具，比如成员管理、通证管理、交易跟踪、数据分析等，帮助 DCO 更好的完成组织运营。

  • 协议

    在基础协议层面，DCO Network 将提供完善的基础设施，如： DCO 创建、管理的协议及 API ；Token 创建、分配、管理的协议及 API ；DCO 激励的智能合约模板等。

DCO Network 帮助 DCO 取得成功

  • 微型组织激励

    学习、吸收和沉淀新技术知识，对每个开发者来说都是不断自我更新的过程。DCO Network 期望通过悬赏创建知识图谱的方式，鼓励开发者自发创建协作整理知识图谱、学习文档、分享学习心得的组织，并构建自己的通证激励方式。

  • 社群组织激励

    DCO Network 将引导传统社群组织基于平台进行通证化改造，协助社群利用通证制定治理规则、激励用户传播参与、持续激励用户活跃等，促进社群高质量产出。对通证激励效果显著的社群，通过将该组织的高阶通证兑换为 DCO 的方式进行激励。

  • 项目团队激励

    DCO Network 期望通过提供完善的基础服务，帮助多方利用通证构建信任和组织，使分散在全球各地的开发者围绕特定的目标结成强有力的、资源高度优化配置的、高效协作的团队，对每个人的贡献予以量化，并且实时、公平的分配收益。
