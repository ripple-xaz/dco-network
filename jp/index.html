Token改变人与人之间的协作关系

DCO分布式协作组织

在中心化组织中，个体通过协议与中心组织建立雇佣关系，协议对个体进行了极大约束，个体失去了很多自由；Token的出现，使得个体间可以通过通证建立关系，并用智能合约记录在区块链中，个体可自由决定协作关系的开始和结束，可同时建立任意数量的协作关系，自由度很高。此外，中心化组织对时间和地域的要求限制了个体的参与；而Token可将不同地域、符合要求的个体组织起来，开展项目协作，资源配置更加优化。这种基于Token建立的组织称为分布式协作组织（Distributed Collaborative Organization，DCO）。

DCO 开启组织管理的新篇

  • 治理

    CODE Network 将提供完善的技术标准、丰富的工具和基础设施，支持 DCO 的创建、运行和发展。比如一个软件开发的 DCO 可以在 平台上完成团队创建，进行团队日常管理，以新的协作组织模式高效率、高质量的进行项目研发，创造出有价值的产品或服务。

  • 公开

    区块链提供了一个诚实可信的记账技术，DCO通过智能合约进行创建，信息全部上链，DCO的团队构成、经营状况、项目信息公开透明，用户可以获得充分的信息以决定是否加入或退出某个 DCO，投资者也可以获得充分的信息以决定是否投资DCO的项目。

  • 权益

    每一个 DCO 可创建自己的“主通证”（DCO Token，DCOT) 和其他高阶通证（High Order Token, HOT)，使用通证为组织成员的贡献进行公平的衡量，以此为依据进行强有力的经济激励。DCO也可以创建部署一组智能合约，根据治理规则自动管理组织成员的通证分配。

  • 流通

    CODE Network将提供基本的交易能力，撮合 CODE 币与任何一个 DCOT 之间的交易。CODE 币作为整个生态系统的基础货币，将登陆主流交易所，与比特币、以太币等主流加密数字货币建立汇率价格联系，实现全渠道交易的畅通，为平台的所有通证带来平台外的价值。

CODE Network为 DCO 提供全方位支撑

CODE Network致力于打造一个去中心化的、全球性的分布式协作组织基础平台

  • 协议

    CODE Network 将常见智能合约标准化、功能化，用户无需任何编程即可快速完成DCO创建、通证发行、权益分配等，并支持DCO通过智能合约创建自己的治理体制和具体规则

  • 工具

    CODE Network 将为DCO提供各种便捷的组织管理工具，比如成员管理、成员邀请、通证管理、交易跟踪、数据分析等，帮助DCO更好的完成组织运营

  • 服务

    CODE Network将对接IT领域常见第三方平台，接入第三方服务。同时消除跨平台协作的障碍，为DCO提供优质、高效的跨平台服务体验，提升DCO协作和管理效率

CODE Network是帮助 DCO 建立信任

  • 项目协作

    DCO利用平台的基础设施、工具和服务，进行新的协作，创造有价值的产品或服务。

  • 资本服务

    CODE Network平台的不断发展将会吸引投资者的目光，为优质项目带来融资机会。

  • 生态建设

    CODE Network生态将持续进化，DCO可通过为平台用户提供服务来获取价值回报。
